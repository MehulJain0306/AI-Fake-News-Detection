# 🛡️ AI Fake News Detection

An AI-powered Fake News Detection platform that analyzes news articles using Machine Learning and Natural Language Processing (NLP).

---

## ✨ Features

- Modern Landing Page
- User Authentication
- Dashboard
- Fake News Detection
- Confidence Score
- News Analysis
- Scan History (Coming Soon)
- Responsive UI
- Dark Theme

---

## 🛠 Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion

### Backend
- Express.js
- FastAPI
- Python

### Machine Learning
- Scikit-learn
- NLP
- TF-IDF
- Logistic Regression
